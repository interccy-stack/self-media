# 📡 智媒体 (Self-Media) v1.2.0

**智媒体** 是 QwenPaw 插件，以卡牌式界面展示云服务器上的所有 AI 智能体，支持一键生成分享链接和直接体验对话。

## ✨ 特性

| 特性 | 说明 |
|------|------|
| 🎴 **卡牌展示** | 每个智能体以渐变配色卡牌呈现，直观美观 |
| 🔄 **自动刷新** | 每 5 分钟自动更新智能体在线状态 |
| 🔗 **自动生成链接** | 点击按钮时自动为智能体生成分享链接 |
| 📋 **一键复制** | 复制分享链接给好友，对方可直接对话 |
| 🚀 **体验对话** | 点击直接打开对话页面（每次都是新会话） |
| 🔍 **搜索筛选** | 支持按名称、ID 或描述搜索智能体 |
| 🌙 **暗黑模式** | 支持明暗主题切换 |
| ☁️ **实时同步** | 云服务器新增智能体后自动显示，无需手动更新 |
| ❓ **使用指南** | 内置新手教学，新手快速上手 |

## 🚀 快速开始

### 前置条件

| 环境 | 要求 |
|------|------|
| **Python** | 3.8+ |
| **Python 依赖** | `paramiko`, `requests` |
| **本地 QwenPaw** | v1.1.10+ |
| **云服务器** | 已部署并运行 QwenPaw + agent-share 插件 |

### 安装步骤

#### 步骤 1：安装依赖

```bash
# 安装 Python 依赖
pip install paramiko requests
```

#### 步骤 2：获取插件

下载并解压 `self-media.zip`

#### 步骤 3：安装插件

```bash
cd /path/to/self-media
qwenpaw plugin install . --force
```

#### 步骤 4：使用

1. 刷新 QwenPaw 页面
2. 在左侧菜单找到「📡 智媒体」
3. 点击进入即可看到智能体列表

### 使用说明

- **默认连接**：插件预设连接作者服务器，开箱即用
- **自定义服务器**：如需连接自己的服务器，创建 `config.json` 配置
- **智能体同步**：云服务器新增/删除智能体后，插件会自动更新显示
- **分享链接**：每次点击都会生成带时间戳的新链接，确保独立会话

## ⚙️ 高级配置

### 连接自己的云服务器

在插件目录创建 `config.json`：

```json
{
  "host": "your-server.com",
  "port": 22,
  "user": "root",
  "password": "your-password",
  "share_domain": "your-domain.com",
  "agent_port": 8889,
  "api_port": 8890,
  "cache_ttl": 120
}
```

### 云服务器部署

确保云服务器已：
1. 安装 QwenPaw
2. 安装 agent-share 插件
3. 启动 cloud-api.py（端口 8890）

```bash
# 在云服务器上启动 cloud-api.py
cd /root/.copaw/plugins/self-media
nohup python3 cloud-api.py > cloud-api.log 2>&1 &
```

### 端口说明

| 端口 | 用途 |
|------|------|
| 22 | SSH 远程连接 |
| 8889 | QwenPaw API 端口 |
| 8890 | cloud-api.py HTTP API（插件通过此端口获取智能体列表） |
| 8088 | 本地 QwenPaw 端口（默认） |

## ❓ 常见问题

### Q: 智能体列表为空怎么办？

A: 检查：
1. 云服务器 cloud-api.py 是否运行（端口 8890）
2. 网络能否访问云服务器

### Q: 点击按钮没反应？

A: 正在生成链接，请稍候（约 3-5 秒）

### Q: 分享链接打不开？

A: 检查云服务器 8889 端口是否开放

### Q: 云服务器新增智能体后会显示吗？

A: ✅ 会自动同步！插件每 5 分钟自动刷新，也可用「🔄 刷新」按钮手动更新
